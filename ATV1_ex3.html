Qual e a diferença de linguagem de programação server side e client side?
Cite exemplos de cada tipo:

  • Linguagem Client-side: É a linguagem que opera no dispositivo do usuário (navegador, aplicativo, etc.), determinando a interface, links, imagens, entre outros.
    Exemplos: Línguagem HTML e JavaScript, que marcam os itens na interface, CSS, Sass, Less que servem para estilizar a interface.

  • Linguagem Server-side: É a linguagem que opera no servidor do dono do site, onde ficam armazenados os dados do cliente, e o usuário não tem acesso ao servidor.
    Exemplos: Linguagem Python, PHP, Java, Node.js
